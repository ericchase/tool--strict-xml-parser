## About

https://github.com/ericchase/tool--strict-xml-parser

A custom parser for a strict subgrammar of XML 1.0. Not meant for standard XML 1.0.

## Build

You will need the `bundle@tsc-standalone` project in order to run the build script. It's currently not available on GitHub, so message me if you want to setup for building.

```bash
bun install
bun run build
```
